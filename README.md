## Getting Started

### Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/Ukhang/cc-vanillaJS.git
   ```
2. Navigate to the project directory:
   ```sh
   cd cc-vanillaJS
   ```
3. Open `index.html` or any specific `.html` file in your browser to explore the visualizations.
